HTML: Hyper Text Markup Language

  • HyperText: Create links connecting web pages together.
  • Markup Language: Use tags to give content meaning and structure.

Install Required Software

  • Visual Studio Code: Download Here
  • Extensions:
    • Live Preview / Live Server
    • Prettier

1. Headings (<h1> to <h6>)

  • Define titles and subtitles.
  • Example: 
    <h1>Heading 1</h1>
<h2>Heading 2</h2>
...
<h6>Heading 6</h6>
  • Purpose: structure content for humans and machines.

2. Paragraph (<p>)

  • Groups sentences and blocks of text.
  • Example: 
    <p>This is a paragraph.</p>
<p>This is another paragraph.</p>

3. Horizontal Rule (<hr>)

  • Defines a thematic break / separates content.
  • Example: 
    <p>Topic 1</p>
<hr>
<p>Topic 2</p>

4. Line Break (<br>)

  • Creates a single line break.
  • Example: 
    <p>Line 1<br>
Line 2</p>

5. List Elements (<ul>, <ol>, <li>)

  • Unordered list (<ul>) → bullets
  • Ordered list (<ol>) → numbers
  • List item (<li>) → each element in a list
  • Example: 
    <ul>
  <li>Tea Bag</li>
  <li>Water</li>
</ul>

<ol>
  <li>Boil water</li>
  <li>Add tea</li>
</ol>

6. Anchor Tag (<a>)

  • Creates hyperlinks.
  • Example: 
    <a href="https://www.coderarmy.in/" target="_blank">Visit Coder Army</a>
  • Target attribute:
    • _self → open in same tab
    • _blank → open in new tab

7. Image Tag (<img>)

  • Embeds an image on the page.
  • Example: 
    <img src="IMAGE_LINK_HERE" alt="Description" width="300" height="300">
  • Replace IMAGE_LINK_HERE with your actual image URL.
