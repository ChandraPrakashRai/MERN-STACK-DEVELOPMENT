Welcome to Coder Hub – a simple multipage HTML project designed to help you learn HTML from scratch. This website contains sequenced lecture pages (Lecture 1 → Lecture 6). Each lecture page has:

  1. One YouTube video link for better understanding.
  2. Detailed notes with headings and points.
  3. Optional images/diagrams for clarity.

HTML Properties Overview Indepth content is available in the LECTURE wise pages

These are just for overview that what we will gonna cover in the series

Below is a summary table that reflects the most important HTML elements & attributes you will learn throughout these lectures:

Category Tags / Elements Description Example
Basic Structure <!DOCTYPE>, <html>, <head>, <body> Defines document type and page structure <html lang="en"> ... </html>
Headings & Text <h1> → <h6>, <p>, <br>, <hr> Headings, paragraphs, line break, horizontal rule <h2>Hello World</h2>
Formatting <b>, <i>, <u>, <strong>, <em>, <mark> Bold, italic, underline, emphasis <strong>Important</strong>
Lists <ul>, <ol>, <li>, <dl> Unordered, ordered, and description lists <ul><li>Item</li></ul>
Links <a href=""> Creates hyperlinks <a href="https://google.com">Google</a>
Images & Media <img>, <audio>, <video> Insert images, audio, video <img src="img.jpg" alt="photo">
Tables <table>, <tr>, <td>, <th> Displays data in rows and columns <table border="1">...</table>
Forms <form>, <input>, <textarea>, <button> Collects user input <input type="text" placeholder="Name">
Semantic Tags <header>, <nav>, <main>, <section>, <footer> Improves page structure and accessibility <footer>End of page</footer>
Miscellaneous <iframe>, <meta>, <title>, <script> Embed content, metadata, page title, scripts <title>My Page</title>
