HTML : Hyper Text Markup Language

HTML stands for HyperText Markup Language. It is the standard language used to create and structure the content of a webpage.

  • HyperText: Connect web pages to one another, forming a "web" of information.
  • Markup Language: Use of tags to structure and give meaning to content.

Lecture 3 Video

1. Headings (<h1> to <h6>)

Headings create hierarchy for your webpage:

<h1>My Daily Routine</h1>
<h2>Morning Tasks</h2>
<h3>Exercise</h3>
<h3>Breakfast</h3>
<h2>Work Tasks</h2>
<h3>Emails</h3>
<h3>Meetings</h3>
  • <h1>: Main page title
  • <h2>: Section title
  • <h3> - <h6>: Subsections of decreasing importance

2. Paragraphs (<p>)

Used to group sentences about a single topic:

<p>I start my day with a short workout and a healthy breakfast.</p>
<p>Later, I focus on coding practice and completing work assignments.</p>

3. Horizontal Rule (<hr>)

Used to separate sections:

<p>Morning Routine</p>
<hr />
<p>Work Tasks</p>

4. Line Break (<br>)

For breaking lines within content:

<p>Home Address:


5. Lists (<ul>, <ol>, <li>)


Organize daily tasks in lists:



<ul>
  <li>Buy groceries</li>
  <li>Walk the dog</li>
  <li>Call a friend</li>
  <li>Read a book</li>
</ul>

<ol>
  <li>Wake up at 6 AM</li>
  <li>Exercise for 30 minutes</li>
  <li>Have breakfast</li>
  <li>Start work/study</li>
</ol>




6. Anchor Tag (<a>)


To create clickable links:



<a href="https://www.coderarmy.in/" target="_blank">Visit Coder Army</a>
