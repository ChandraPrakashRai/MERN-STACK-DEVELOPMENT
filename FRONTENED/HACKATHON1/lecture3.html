HTML Nested Lists & Tables: The Complete Guide

In this lecture, we will explore two very important concepts of HTML: Nested Lists and Tables. These elements help us structure data in a clean, readable way.

    1. Introduction to Nested Lists

    Nested lists mean placing one list inside another list. HTML supports both unordered (bullets) and ordered (numbers/letters) lists. 

    <ul>
  <li>Fruits
    <ul>
      <li>Apple</li>
      <li>Banana</li>
      <li>Mango</li>
    </ul>
  </li>
  <li>Vegetables
    <ul>
      <li>Carrot</li>
      <li>Broccoli</li>
    </ul>
  </li>
</ul>
    output of the above code:
    • Fruits
      • Apple
      • Banana
      • Mango
    • Vegetables
      • Carrot
      • Broccoli

    2. Ordered List inside Unordered List

    <ul>
  <li>Steps to Bake Cake:
    <ol>
      <li>Mix ingredients</li>
      <li>Preheat oven</li>
      <li>Bake for 30 minutes</li>
    </ol>
  </li>
</ul>

    output for the above code example:

    • Steps to Bake Cake:
      1. Mix ingredients
      2. Preheat oven
      3. Bake for 30 minutes

    4. Introduction to Tables

    Tables allow us to arrange data in rows and columns. A table is created using: - <table> → main container
    - <tr> → table row
    - <th> → table heading
    - <td> → table data 

    <table border="1">
  <tr>
    <th>Name</th>
    <th>Age</th>
    <th>City</th>
  </tr>
  <tr>
    <td>John</td>
    <td>25</td>
    <td>New York</td>
  </tr>
  <tr>
    <td>Alice</td>
    <td>22</td>
    <td>London</td>
  </tr>
</table>
    Name Age City
    Chandra Prakash 21 New Delhi
    Alice 22 London

    9. Combining Lists & Tables

    We can combine both for advanced layouts. Example: Table showing categories with sub-items in a list. 

    <table border="1">
  <tr><th>Category</th><th>Items</th></tr>
  <tr>
    <td>Fruits</td>
    <td>
      <ul>
        <li>Apple</li>
        <li>Banana</li>
      </ul>
    </td>
  </tr>
</table>
