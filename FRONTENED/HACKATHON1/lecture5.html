Lecture 5: Forms in HTML

14. Lecture Video

Watch the detailed video here:

1. Introduction

Forms in HTML are used to collect information from users. Without forms, websites would only show information but could not take any input.

Examples where forms are used daily:

  • Google search box
  • Login/Signup page
  • Online shopping checkout
  • Feedback forms

2. Basic Form Structure

A form always starts with the <form> tag.

<form>
    ... form elements go here ...
</form>

3. Text Input

Used when user needs to type text (like name, username).

<label for="username">Enter your Name:</label>
<input type="text" id="username" name="username">

Example Use: Registration form asking for your name.

4. Password Input

Hides the characters entered (used in login forms).

<label for="password">Password:</label>
<input type="password" id="password" name="password">

5. Email Input

Validates email format automatically.

<label for="email">Email:</label>
<input type="email" id="email" name="email">

6. Radio Buttons

Used when only one choice is allowed (example: gender, payment method).

<p>Select Gender:</p>
<input type="radio" id="male" name="gender" value="male">
<label for="male">Male</label>

<input type="radio" id="female" name="gender" value="female">
<label for="female">Female</label>

7. Checkboxes

Used when multiple selections are possible (example: hobbies, food preferences).

<p>Select Hobbies:</p>
<input type="checkbox" id="cricket" name="hobby" value="cricket">
<label for="cricket">Cricket</label>

<input type="checkbox" id="music" name="hobby" value="music">
<label for="music">Music</label>

<input type="checkbox" id="reading" name="hobby" value="reading">
<label for="reading">Reading</label>

8. Textarea

For longer text like messages, feedback, comments.

<label for="feedback">Your Feedback:</label><br>
<textarea id="feedback" name="feedback" rows="4" cols="40"></textarea>

9. Dropdown (Select)

For choosing one option from many.

<label for="country">Select Country:</label><br>
<select id="country" name="country">
    <option value="">--Select--</option>
    <option value="in">India</option>
    <option value="us">USA</option>
    <option value="uk">UK</option>
</select>

10. Other Useful Inputs

  • type="number" → Age, Quantity
  • type="date" → Birthday, Appointment
  • type="file" → Upload documents
  • type="color" → Choose theme color
  • type="range" → Select volume or brightness
<label for="age">Enter Age:</label>
<input type="number" id="age" name="age">

11. Submit Button

Sends data from the form to the server.

<input type="submit" value="Submit Form">

12. Real-Life Example: Login Form

<form>
    <label for="email">Email:</label>
    <input type="email" id="email" name="email">
    <br><br>

    <label for="password">Password:</label>
    <input type="password" id="password" name="password">
    <br><br>

    <input type="submit" value="Login">
</form>

13. Summary

  • Form = Collect user input
  • Text/Password/Email = Basic fields
  • Radio = One choice
  • Checkbox = Multiple choices
  • Textarea = Long text
  • Dropdown = Select from list
  • Submit = Send data
